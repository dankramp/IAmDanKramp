JavaScript Projects

Springy Graph

Generate random graphs and watch them as they settle into place via spring force algorithms!

Most settings are customizable, such as the number of points and edges, as well as the spring length (in pixels) used for calculating the forces. This application also lets you interact with the graph using your mouse or touch screen!

Pic2Text

Create a textual representation of your own uploaded image file with just the push of a button!

This application is extremely easy and quick to use. It also allows you to simply modify the resolution of the outputted image and copy the resulting text to your clipboard!

HeatMapper.js

Create a HeatMapper JavaScript object to easily map values directly to a color ramp.

Utilize this browser-based interface to visualize your heat map and generate the code automatically. Useful for developers who want to create a dynamic color spectrum based on variable input values. For example, inputting a rainfall measurement in inches and returning a color that it corresponds to in order to create a 2D map of rainfall.
